Privacy

We store nothing.

ShadoLogiq Labs sites are static pages served from GitHub Pages. There is no server we run, no database, no analytics, and no cookies. This page exists to say that plainly.

✓ 0 cookies· ✓ 0 trackers· ✓ 0 analytics· ✓ no account

What we collect

Nothing. We set no cookies, run no analytics scripts, and embed no third-party trackers. We do not ask you to log in, and there is no account to create. Nothing you do on these pages is recorded by us, because there is no "us" in the data path — the pages are static files your browser downloads and runs locally.

What your browser does on its own

These sites are hosted on GitHub Pages. When your browser requests a page, GitHub's servers receive that request and may log standard technical information (such as your IP address and user-agent) the way any web host does. That logging is governed by GitHub's privacy statement, not by us, and we never see or receive it.

Brand assets (logos, icons) and embeds load from shadologiq.com — the same trust boundary. The tools we build, like Gapps Embed, run entirely in your browser and make no network calls beyond loading their own static files.

Third-party tools in the catalog

Our catalog links to tools and apps made by other people. Each one has its own privacy posture, which we document in its catalog entry — including telemetry and account requirements — so you can decide before you install. Following a link to one of those tools takes you off our sites and under their terms.

Changes

If this ever changes — if we ever add anything that touches your data — we will say so here first, in plain language, before it ships. The "no build step, no telemetry, client-side everything" stance is the brand, not a phase.

Questions

Open an issue or discussion at github.com/shadologiq-labs. That's the contact channel for everything.
